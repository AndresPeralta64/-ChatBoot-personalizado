# -ChatBoot-personalizado
En el siguiente archivo HTML se evidencia la implementación de un chatbot educativo personalizado, integrado en una aplicación web. Dicho asistente virtual ha sido entrenado a partir de una base de conocimiento propia que contiene información básica sobre mecánica cuántica, incluyendo su origen, definición y conceptos fundamentales, lo que le permite interactuar con el usuario de manera informativa y contextualizada.

Para la ejecución de este proyecto se necesita descargar todo siguiendo la estructura de este repositorio, posterior a eso se ejecuta (doble clic) al archivo .bat.
